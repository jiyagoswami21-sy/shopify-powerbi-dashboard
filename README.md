# Shopify Sales & Customer Funnel Dashboard

This Power BI dashboard analyzes Shopify sales performance and customer behavior.

## Key Features
- Net Sales, Profit, Average Order Value
- Repeat Rate, Lifetime Value
- Product Type & Gateway breakdown
- Geographic customer distribution

## Tools Used
- Power BI
- DAX Measures
- Power Query
- Excel Dataset

## Files Included
- Dashboard (.pbix)
- Dataset (.xlsx)
- Screenshot (.png)
